## 2026-05-25

### Added

**Phrases (references/phrases.md)**
- "worth [verb-ing]" family expanded inside throat-clearing: "worth knowing / understanding / bringing up / being honest about / flagging / knowing about going in"
- Filler additions: "In practice,", "In reality,", "Ultimately,", "When all is said and done", "What this means in practice is"
- Performative Emphasis: "Trust me on this", "Believe me", "I'm telling you"
- **New section: Performative Authority**
  - "honest" as adjective ("honest tradeoffs", "honest assumption", "the honest middle is", "I'll be honest about")
  - "real" as credibility marker (distinct from "real" meaning non-fictional)
  - "X is real" declarations (validation through assertion)
- **New section: Validating the Audience** — "you're not alone", "this is normal", "that's okay", "X is fine", and similar reassurance tics. One earns its place; stacked it reads as cloying.

**Structures (references/structures.md)**
- **New section: Headers** — explicit rule against Wh-headers ("What helps", "Why this matters", "How X works", question-shaped headers), with fix shapes.
- **New section: Symmetric Structures** — AI loves balance; real human writing is asymmetric. Pros/Cons with matching counts, three-bullet groups with identical sentence shape, parallel-clause sentences.
- Em-dash rule strengthened: the fix is usually two sentences, not a colon or parens swap.

**SKILL.md**
- Quick Checks expanded: Wh-headers, "honest X" / "real X" markers, "X is real" / "this is normal" validation tics, "worth [verb-ing]", three-bullets-same-shape, symmetric two-sided lists.
- Scoring matrix split Directness into Directness + Specificity (separate failure modes); threshold rescaled from 35/50 to 42/60.
