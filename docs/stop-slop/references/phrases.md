# Phrases to Remove

## Throat-Clearing Openers

Remove these announcement phrases. State the content directly.

- "Here's the thing:"
- "Here's what [X]"
- "Here's this [X]"
- "Here's that [X]"
- "Here's why [X]"
- "The uncomfortable truth is"
- "It turns out"
- "The real [X] is"
- "Let me be clear"
- "The truth is,"
- "I'll say it again:"
- "I'm going to be honest"
- "Can we talk about"
- "Here's what I find interesting"
- "Here's the problem though"
- "Here's the kicker"
- "Here's where it gets interesting"
- "Here's what most people miss"
- "Here's the deal"

Any "here's what/this/that" construction is throat-clearing before the point. Cut it and state the point.

## Emphasis Crutches

These add no meaning. Delete them.

- "Full stop." / "Period."
- "Let that sink in."
- "This matters because"
- "Make no mistake"
- "Here's why that matters"

## Business Jargon

Replace with plain language.

| Avoid | Use instead |
|-------|-------------|
| Navigate (challenges) | Handle, address |
| Unpack (analysis) | Explain, examine |
| Lean into | Accept, embrace |
| Landscape (context) | Situation, field |
| Game-changer | Significant, important |
| Double down | Commit, increase |
| Deep dive | Analysis, examination |
| Take a step back | Reconsider |
| Moving forward | Next, from now |
| Circle back | Return to, revisit |
| On the same page | Aligned, agreed |

## Adverbs

Kill all adverbs. No -ly words. No softeners, no intensifiers, no hedges.

Specific offenders:

- "really"
- "just"
- "literally"
- "genuinely"
- "honestly"
- "simply"
- "actually"
- "deeply"
- "truly"
- "fundamentally"
- "inherently"
- "inevitably"
- "interestingly"
- "importantly"
- "crucially"

Also cut these filler phrases:

- "At its core"
- "In today's [X]"
- "It's worth noting" / "worth knowing" / "worth understanding" / "worth bringing up" / "worth being honest about" / "worth flagging" / "worth knowing about going in"
- "At the end of the day"
- "When it comes to"
- "In a world where"
- "The reality is"
- "In practice," / "In reality," / "What this means in practice is"
- "Ultimately," / "When all is said and done"

Any "worth [verb-ing]" construction is announcing credibility before earning it. Cut and state the thing.

## Banned Sentence Openers

These announce themselves before saying anything. Cut and rejoin to the previous sentence, or start with the actual content.

- "Certainly,"
- "Absolutely,"
- "Indeed,"
- "Moreover,"
- "Furthermore,"
- "Additionally,"
- "Interestingly,"
- "Notably,"
- "Importantly,"
- "Crucially,"
- "Ultimately,"
- "Overall," (as paragraph starter)
- "In conclusion,"
- "Firstly," / "Secondly," / "Thirdly,"
- "Great question!" / "That's a great point!"
- "Sure," / "Of course," (response openers)
- "I'd be happy to..."
- "Look,"
- "So," (paragraph starter)

Most of these signal AI's "let me organize this for you" reflex. The writing should flow without them.

## Meta-Commentary

Remove self-referential asides. The essay should move, not announce its own structure.

- "Hint:"
- "Plot twist:" / "Spoiler:"
- "You already know this, but"
- "But that's another post"
- "X is a feature, not a bug"
- "Dressed up as"
- "The rest of this essay explains..."
- "Let me walk you through..."
- "In this section, we'll..."
- "As we'll see..."
- "I want to explore..."

## Performative Emphasis

False intimacy or manufactured sincerity:

- "creeps in"
- "I promise"
- "They exist, I promise"
- "Trust me on this" / "Believe me" / "I'm telling you"

## Performative Authority

Words that *announce* credibility instead of *being* credible. AI reaches for these to sound trustworthy.

**"Honest" as adjective.** "Honestly" is already on the adverb kill list, but the adjective is everywhere:

- "Honest tradeoffs:"
- "Honest assumption"
- "The honest middle is"
- "I'll be honest about [X]"
- "Honest truth"

If the rest of the sentence is honest, you don't need to advertise it. Cut "honest" and ship the claim.

**"Real" as a credibility marker.** Different from "real" meaning "not fictional." When "real" is doing emphasis work:

- "a real choice" → "a choice"
- "the real question is" → state the question
- "real options" → "options"
- "real money on the table"
- "the real problem"
- "really matters" (already covered as adverb, but the adjective form is sneakier)

If removing "real" doesn't change the meaning, remove it.

**"X is real" declarations.** Asserting the existence of something instead of describing it:

- "Withdrawal is real."
- "The risk is real."
- "Cravings are real."
- "Precipitated withdrawal is real."

One use can land when the reader genuinely needs the existence asserted (someone in denial). Stacked through a page, the pattern reads as performed reassurance. Default: cut and describe what the thing *does*.

## Validating the Audience

Reassurance-shaped sentences AI sprinkles into "supportive" content. Sometimes load-bearing; usually slop when stacked.

- "You're not alone."
- "You're not making it up."
- "This is normal." / "That's normal."
- "You're in normal company."
- "That's okay." / "And that's okay."
- "[X] is fine."
- "It's not your fault."

One earns its place per page when a reader genuinely needs validating (recovery content, mental health, anything addressing shame). Two or more in close proximity reads as cloying. Watch especially for the "X, that's not failure" / "Y, you're not weak" pattern at the end of paragraphs — that's the AI signature.

**Test:** if you can remove the sentence and the rest of the paragraph still does the work, remove it.

## Pedagogical Hand-Holding

The teacher-student voice. AI defaults to this whenever it's uncertain whether the reader knows something. The result is a condescending walk-through of obvious points.

- "Let's break this down"
- "Let's unpack this" / "Let me unpack"
- "Let's explore"
- "Let's dive in" / "Let's delve into"
- "Think of it as..."
- "Think of it like..."
- "Imagine a world where..."
- "Suppose for a moment that..."
- "Walk with me through..."

Cut them and state the thing. Or if the reader genuinely needs an analogy, give the analogy without the wind-up.

## AI Vocabulary Tells

Words that became statistically overrepresented in AI-generated text after late-2022. Avoid or replace.

| Word | Replacement |
|------|-------------|
| delve / delves / delving | examine, look at, explore |
| tapestry | mix, combination, range |
| landscape (figurative) | field, situation |
| nuanced (as filler) | complex, subtle, specific — or cut |
| robust (outside engineering) | strong, solid |
| leverage (verb) | use |
| utilize | use |
| facilitate | help, enable, make easier |
| streamline | simplify |
| harness | use, apply |
| paradigm | model, approach |
| synergy / synergies | cooperation, combined effect |
| ecosystem (figurative) | system, community |
| pivotal | central, key |
| intricate / meticulous | complex, careful |
| bolster | strengthen, support |
| garner | gain, earn |
| underscore | emphasize, show |
| interplay | interaction |
| foster | encourage |
| transformative | a real change |
| seamless / seamlessly | smooth (or cut) |
| testament (as in "a testament to") | evidence, proof |
| vibrant | active, lively (usually cut) |
| certainly (sentence-initial) | usually deletable |

If reaching for one of these, replace with the concrete word or cut the phrase.

## The "Serves As" Dodge

Replacing simple `is` or `are` with pompous alternatives. AI avoids basic copulas because its repetition penalty pushes toward fancier verbs.

- "X serves as Y" → "X is Y"
- "X stands as Y" → "X is Y"
- "X marks Y" → "X is Y" (unless something genuinely *marked* a moment)
- "X represents Y" → "X is Y" (unless symbolism is the actual point)
- "X embodies Y" → "X is Y"
- "X functions as Y" → "X is Y"

Default to `is`. Reach for the fancier verb only when it does specific work the copula can't.

## Telling Instead of Showing

Announcing difficulty or significance rather than demonstrating it:

- "This is genuinely hard"
- "This is what leadership actually looks like"
- "This is what X actually looks like"
- "actually matters"

## Vague Declaratives

Sentences that announce importance without naming the specific thing. Kill these.

- "The reasons are structural"
- "The implications are significant"
- "This is the deepest problem"
- "The stakes are high"
- "The consequences are real"

If a sentence says something is important/deep/structural without showing the specific thing, cut it or replace it with the specific thing.
