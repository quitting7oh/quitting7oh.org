/* font: load Fraunces (display) + keep your body font for .org */

.logo-link   { display:inline-flex; align-items:center; gap:.55em; text-decoration:none; }

               font-weight:600; font-size:1.5rem; letter-spacing:-.01em;
               line-height:1; color:var(--foreground); }
.logo-wm b   { font-weight:600; color:var(--primary); }     /* the 7 */
.logo-wm i   { font-family: var(--font-sans, system-ui, sans-serif);
               font-style:normal; font-weight:500; font-size:.45em;
               color:var(--muted-foreground); margin-left:.05em; }

Inline the SVG (don't use <img>) so it can read the theme variables. The mark and wordmark both inherit --foreground / --primary from whatever data-theme + mode is active, so a single component covers all 16 states.
